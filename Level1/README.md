## 핸드폰 번호 가리기

```
charAt()란 무엇인가? charAt 에 대해서 알고 넘어가자. String으로 저장된 문자열 중에서 특정 문자를 선택해서 char 타입으로 변환해준다.
기본 형태 -> String sample = "abc"; char target = sample.charAt(0);
charAt(i)
i 자리에는 int 형 변수를 넣어서 원하는 위치의 문자를 가져올 수 있다. (i번째 자리에 있는 문자을 가져올 수 있다.)
```
